Here you can also find code examples of what you need to theme your next AllPlayers related project and make it look consistant with AllPlayers styles.

Compass

Compass is an open-source CSS Authoring Framework. In addition to allowing you to write SASS (and generate CSS) it also has many built in functions that makes CSS generation fast and easy. Please look at the reference doc to view what types of built-in mixins and helpers can make your CSS generation experience faster and more efficient.

In order to compile CSS while you are writing SCSS simply type: 
$ compass watch
into the command-line inside the project folder. If there is a merge conflict in a css file it is best practice to run 
$ compass clean; compass compile;
in order to remove and recompile the CSS.
The Importance of the Gem Bundler

Bundler

Inside of the Basic Foundation theme file there is a Ruby Gem bundle that installs all the necessary gems (AND NECESSARY VERSIONS OF GEMS) that you need to get started with theming in SASS.

Simply type 
$ bundle install
into the command-line while you are inside of the basic_foundation folder and all of the necessary gems will be installed.
Jekyll

All of our brochure sites (including this one) are built using Jekyll.

Simply type 
$ jekyll serve -w
into the command-line while you are inside the project folder in order to compile and serve your files on the localhost. Read more about Jekyll here.

Jekyll is built using the liquid markup language. Visit here to see some basic examples of how this language is used.
