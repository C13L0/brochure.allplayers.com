MySQL Database Administrator

The database administrator (DBA) will be responsible for the performance, integrity and security of the AllPlayers.com databases including:

  • Establishing the needs of users and monitoring user access and security
  • Monitoring performance and managing parameters to provide fast query responses
  • Mapping out the conceptual design for a planned database in outline
  • Controlling access permissions and privileges
  • Developing, managing and testing back-up and recovery plans
  • Ensuring storage, archiving, back-up and recovery procedures are functional
  • Capacity planning
  • Communicating regularly with staff to ensure database integrity and security

Systems Administrator

  • Perform daily system monitoring, verifying the integrity and availability of all hardware, server resources, systems and key processes, reviewing system and application logs, and verifying completion of scheduled jobs such as backups
  • Perform regular security monitoring to identify any possible intrusions
  • Perform daily backup operations, ensuring all required file systems and system data are successfully backed up
  • Perform regular file archival and purge as necessary
  • Create, change, and delete user accounts per request
  • Repair and recover from hardware or software failures
  • Perform periodic performance reporting to support capacity planning
  • Perform ongoing performance tuning, hardware upgrades, and resource optimization as required
